FLUJO DE LA APLICACIÓN
================================================================================

1. INICIO
   └─ Verifica si hay URL guardada
      ├─ Sí → Va a Lobby (auto-conecta)
      └─ No → Muestra URL Config Screen

2. URL CONFIG SCREEN
   ├─ Usuario ingresa URL
   ├─ Se valida formato
   ├─ Se guarda en SharedPreferences
   ├─ Se conecta vía Socket.IO
   └─ Navega a Lobby Screen

3. LOBBY SCREEN
   ├─ Usuario ingresa nickname
   ├─ Emite evento join_lobby
   ├─ Escucha lobby_status
   ├─ Visualiza jugadores en tiempo real
   ├─ Espera que ambos jugadores se unan
   └─ Emite assign_pokemon

4. TEAM SCREEN
   ├─ Recibe 3 Pokémon asignados
   ├─ Muestra información completa
   ├─ Usuario confirma ready
   ├─ Emite evento ready
   ├─ Escucha battle_start
   └─ Auto-navega a Battle Screen

5. BATTLE SCREEN
   ├─ Batalla en tiempo real
   ├─ Sistema de turnos alternados
   ├─ Usuario emite attack cuando es su turno
   ├─ Escucha turn_result
   ├─ Actualiza HP animadamente
   ├─ Escucha battle_end
   ├─ Muestra diálogo de victoria/derrota
   └─ Opción de volver a Lobby

INSTRUCCIONES DE EJECUCIÓN
================================================================================

1. Instalar dependencias:
   $ cd /sessions/upbeat-cool-heisenberg/mnt/pokemon_app
   $ flutter pub get

2. Ejecutar la aplicación:
   $ flutter run

3. Usar la aplicación:

   - Visualiza el resultado cuando la batalla termine
